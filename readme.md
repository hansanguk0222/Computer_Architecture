# Computer_Architecture
컴퓨터 구조론은 CPU, 기억장치, I/O장치에 대해서 깊이 다루는 학문이다.
<br>

## 컴퓨터 구조론을 공부하게 된 이유
CS(Computer Science)가 이론적인 내용뿐이라서 필요없을 줄 알았는데 진짜 겁나 필요하다. 그리고 내가 이론적인 공부에서 끝냈기 때문에 필요 없다고 느꼈던 것이다. 그렇기 때문에 반성하는 마음으로 직접 코드도 짜면서 공부하기로 마음 먹었다.
<br>
